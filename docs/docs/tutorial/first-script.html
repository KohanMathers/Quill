Your First Script

Your First Script

Follow this short guide to write, load, and run your first Quill script on a Minecraft server.

What you'll build

A small welcome script that greets players when they join, gives them a few items, and shows a title. This demonstrates event handlers, basic built-in functions, and a simple toggle command.

Prerequisites

  • Quill installed on your server (see the Installation Guide)
  • A basic familiarity with the Quill syntax (see Language Basics)

1 — Create the script file

Create a new file named welcome.ql (or any name you prefer) and paste the following code:

let welcome_enabled = true;

OnEvent(PlayerJoin) {
    if welcome_enabled {
        sendmessage(player, "Welcome to the server, {player.name}!");
        give(player, "minecraft:bread", 5);
        sendtitle(player, "Welcome!", player.name, 20, 60, 20);
    }
}

OnEvent(PlayerChat) {
    if chat.message == "!toggle_welcome" && is_op(player) {
        welcome_enabled = !welcome_enabled;
        sendmessage(player, "Welcome system: {welcome_enabled}");
    }
}

Notes:
- welcome_enabled is a simple persistent flag.
- OnEvent(PlayerJoin) runs each time a player joins and receives the player context.
- is_op(player) checks operator status before allowing the toggle.

2 — Load and run the script

On your server, use the in-game or console Quill commands:

/quill load welcome.ql

If your server shows errors when running, check the server console/logs for syntax or runtime messages.

3 — Test it

  1. Join the server with a non-OP account and confirm you receive the welcome messages, 5 bread items, and the title.
  2. With an OP account, type !toggle_welcome in chat to toggle the system on/off and confirm the response message.

4 — Extend the script (ideas)

  • Add per-scope behavior: wrap welcome logic so it only applies inside a Scope region (in_region + subscope.players).
  • Track how many players have received the welcome and persist a counter.
  • Add a cooldown per player to avoid spamming returning players.

Troubleshooting

  • If the script fails to load, ensure the file is in the correct scripts directory used by your Quill plugin and check the console for parser errors.
