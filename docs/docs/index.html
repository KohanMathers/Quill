Welcome to Quill

Welcome to Quill

Quill is a sandboxed scripting language for Minecraft event automation. Create community-run events, mini-games, and automated systems without bugging your admins.

What is Quill?

Quill allows players to write custom scripts that react to in-game events within defined physical boundaries. Scripts run in isolated scopes with granular permission controls, making it safe for server owners to delegate creativity to their community.

Key Features

  • Event-Driven Architecture - React to player actions, block changes, and world events in real-time
  • Sandboxed Execution - Scripts run in isolated scopes with physical boundaries and permission controls
  • Player-Friendly Syntax - Simple, readable syntax designed for non-programmers
  • Persistent State - Variables and scope data survive server restarts
  • Web-Based Editor - Write and test code through an in-browser editor
  • Permission System - Admins control which functions each scope can use (whitelist/blacklist mode)

Quick Example

// Create an arena with a simple deathmatch game
let arena = new Scope(-50, 60, -50, 50, 100, 50)
let game_active = false

OnEvent(PlayerChat) {
    if chat.message == "!start" && in_region(player, arena) {
        game_active = true
        broadcast("Game starting!")

        for p in arena.players {
            set_gamemode(p, "survival")
            give(p, "stone_sword")
            heal(p)
        }
    }
}

OnEvent(PlayerDeath) {
    if game_active && in_region(player, arena) {
        broadcast("{player.name} was eliminated!")
    }
}
