API Reference — Scope Functions

API Reference — Scope Functions

This page documents all built-in functions for managing scopes in Quill scripts. Scopes are physical regions in the world where scripts run and maintain their own state.

Scope Management

addtoscope

Add a player to a scope's player list.

addtoscope(player, scope) -> Boolean

Returns true if player was added.

removefromscope

Remove a player from a scope's player list.

removefromscope(player, scope) -> Boolean

Returns true if player was removed.

getplayers

Get all players currently in a scope.

getplayers(scope) -> List

Returns list of Player objects.

Example managing scope membership:

// Add players in region
OnEvent(PlayerMove) {
    if in_region(player, arena) && !contains(arena.players, player) {
        addtoscope(player, arena);
        sendmessage(player, "Welcome to the arena!");
    }
}

// Remove players who leave
OnEvent(PlayerQuit) {
    if contains(arena.players, player) {
        removefromscope(player, arena);
    }
}

Region Checking

in_region

Check if a player or location is within a scope's physical bounds.

in_region(player, scope) -> Boolean
in_region(location, scope) -> Boolean

Returns true if target is inside the scope's region.

Example using region checks:

OnEvent(PlayerDamage) {
    // No PvP outside arena
    if !in_region(player, pvp_zone) {
        cancel(event);
        sendmessage(player, "PvP is only allowed in the arena!");
    }
}

Region Information

get_region

Get a scope's physical boundaries.

get_region(scope) -> Region

Returns Region object with properties:
- x1, y1, z1: Minimum corner
- x2, y2, z2: Maximum corner

set_region

Update a scope's physical boundaries.

set_region(scope, x1, y1, z1, x2, y2, z2) -> Boolean

Returns true if region was updated.

Example using region data:

func teleport_to_random(player, scope) {
    let region = get_region(scope);

    // Random position within bounds
    let x = random(region.x1, region.x2);
    let y = region.y1;  // Ground level
    let z = random(region.z1, region.z2);

    teleport(player, x, y, z);
}

Scope Variables

Scopes can store persistent variables as properties:

// Set scope variables
arena.game_active = false;
arena.min_players = 4;
arena.round = 0;

// Access in events
OnEvent(PlayerJoin) {
    if len(arena.players) >= arena.min_players && !arena.game_active {
        start_game();
    }
}

Variables persist until server restart or scope deletion.

Examples

Complete arena management:

let arena = new Scope(-50, 60, -50, 50, 100, 50);
arena.game_active = false;
arena.min_players = 2;
arena.round = 0;

// Player join/leave handling
OnEvent(PlayerMove) {
    if in_region(player, arena) {
        if !contains(arena.players, player) {
            addtoscope(player, arena);
            sendmessage(player, "Welcome to the arena!");
            check_game_start();
        }
    } else {
        if contains(arena.players, player) {
            removefromscope(player, arena);
            sendmessage(player, "Come back soon!");
            check_game_end();
        }
    }
}

func check_game_start() {
    if !arena.game_active && len(arena.players) >= arena.min_players {
        broadcast("Game starting with {len(arena.players)} players!");
        arena.game_active = true;
        arena.round = arena.round + 1;

        for player in arena.players {
            reset_player(player);
        }
    }
}

func check_game_end() {
    if arena.game_active && len(arena.players) < arena.min_players {
        broadcast("Not enough players, ending game.");
        arena.game_active = false;

        for player in arena.players {
            set_gamemode(player, "adventure");
        }
    }
}

Multi-scope interaction:

let lobby = new Scope(-10, 60, -10, 10, 70, 10);
let game = new Scope(-50, 60, -50, 50, 100, 50);

func transfer_players(from_scope, to_scope) {
    for player in from_scope.players {
        removefromscope(player, from_scope);
        addtoscope(player, to_scope);

        // Teleport to new scope
        let region = get_region(to_scope);
        teleport(player, 
            (region.x1 + region.x2) / 2,
            region.y1,
            (region.z1 + region.z2) / 2
        );
    }
}

// Example usage
OnEvent(PlayerChat) {
    if chat.message == "!start" && in_region(player, lobby) {
        transfer_players(lobby, game);
        broadcast("Game starting!");
    }
}
