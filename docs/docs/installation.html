Installation

This page shows how to install and configure Quill on your Minecraft server.

Requirements

  • Minecraft server: Bukkit / Spigot / Paper
  • Java 21 or higher

Quick setup

  1. Download the latest Quill.jar-0.1.0-ALPHA from the Releases page.
  2. Place Quil-0.1.0-ALPHAl.jar in your server's plugins/ folder.
  3. Restart your Minecraft server so the plugin can load.
  4. Configure Quill using the plugin config (example path: plugins/Quill/config.yml).

Detailed steps

1) Download

Get the most recent release from the project's Releases page (for example: GitHub Releases). Save the Quill.jar file to a location accessible from your server machine.

2) Install

Copy Quill-0.1.0-ALPHA.jar into your server's plugins/ directory. On most hosting setups this is in the server root.

3) Start the server

Restart your server (or run a plugin reload if your host supports it) so the plugin initializes and creates its configuration files and data folders.

4) Configure

Open the generated plugins/Quill/config.yml to adjust runtime options.

Common Commands (Admin)

These commands are available via the Quill plugin:

/quill scope create <scope_name> <owner_username> <x1> <y1> <z1> <x2> <y2> <z2>
/quill scope join <scope_name> <username>
/quill permission revoke <scope_name> <function_name>
/quill permission grant <scope_name> <function_name>
/quill permission list <scope_name>

Using Quill (Player)

Players can edit and run scripts using in-game commands:

/quill edit        # Open in-game editor
/quill load <file> # Load a script file
/quill run         # Compile/run current script
/quill info        # Show scope info

Notes and best practices

  • Quill runs scripts in sandboxed scopes with per-scope permission controls. Use the permission commands to revoke or grant built-in functions for a scope.
  • Persisted state (variables and scope data) is stored by the plugin — make sure to backup the plugin folder when migrating servers.
  • If you rely on advanced features described in the SPEC (for example, custom events or persistence), verify your config.yml allows them and review permission settings.

Troubleshooting

  • If the plugin does not load, check the server logs for Java exceptions. Ensure you run Java 21+.
  • If features appear missing, confirm you installed the correct Quill-0.1.0-ALPHA.jar release and restart the server after replacing the jar.
