Quick Start

Quick Start

Welcome to Quill! This guide will help you write your first script and get familiar with the basics.

Introduction

This guide will help you set up your first script and understand the basics of Quill. It assumes you have read the introduction to Quill and have installed Quill on your server.

Your First Script

Let's create a simple welcome-message script that greets players when they join.

Step 1: Create a Scope

First, create a scope for your script. A scope defines where your script can operate:

/quill scope create welcome-zone YourUsername -100 60 -100 100 120 100 whitelist

This creates a scope named welcome-zone that:

  • Covers coordinates from (-100, 60, -100) to (100, 120, 100)
  • Is owned by you
  • Uses whitelist mode (only explicitly granted functions work)

Allow the scope to send messages to players:

/quill scope permission grant welcome-zone sendmessage

Open the web editor:

/quill edit welcome-zone/welcome.ql

Click the link that appears in chat. This opens the Quill editor in your browser.

Step 2: Write a Simple Script

In the editor, write the following script:

OnEvent(PlayerJoin) {
    // Get the player's username
    username = player.name

    // Send the player a welcome message
    sendmessage(player, "Welcome to the server, " + username + "!")
}

Press the Save & Exit button in the top right.

Step 3: Load the Script

In-game, load the script with:

/quill load welcome-zone, welcome.ql

This loads and runs the script.

Step 4: Test It Out

Have a player join the game to see if they receive the message. If they do, congratulations — you’ve just created your first script!

Basic Examples

Example 1: Welcome Message

OnEvent(PlayerJoin) {
    // Get the player's username
    username = player.name

    // Send the player a welcome message
    sendmessage(player, "Welcome to the server, " + username + "!")
}

Example 2: Heal on Command

OnEvent(PlayerChat) {
    if (chat.message == "!heal") {
        heal(player)
    }
}

Example 3: Block Break Counter

let blocks_broken = 0

OnEvent(BlockBreak) {
    blocks_broken = blocks_broken + 1
    sendmessage(player, "You have broken " + blocks_broken + " blocks!")
}
