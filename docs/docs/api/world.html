API Reference — World Functions

API Reference — World Functions

This page documents all built-in functions for modifying the Minecraft world in Quill scripts. These functions let you manipulate blocks, spawn entities, and control environmental effects.

Block Management

set_block

Place or replace a block at a location.

set_block(x, y, z, block_id) -> Boolean
set_block(location, block_id) -> Boolean
  • block_id: Minecraft block ID (e.g., "minecraft:stone")
    Returns true if block was set.

Example:

// Set block at coordinates
set_block(0, 65, 0, "minecraft:diamond_block");

// Set block at location
let pos = location(0, 65, 0);
set_block(pos, "minecraft:stone");

get_block

Get the block type at a location.

get_block(x, y, z) -> String
get_block(location) -> String

Returns block ID as string.

break_block

Break a block naturally (drops items).

break_block(x, y, z) -> Boolean
break_block(location) -> Boolean

Returns true if block was broken.

Example building a structure:

func build_platform(x, y, z, size) {
    for dx in range(-size, size) {
        for dz in range(-size, size) {
            set_block(x + dx, y, z + dz, "minecraft:stone");
        }
    }
}

Entity Management

spawn_entity

Create a new entity in the world.

spawn_entity(entity_type, x, y, z) -> Entity
spawn_entity(entity_type, location) -> Entity
  • entity_type: Minecraft entity ID (e.g., "minecraft:zombie")
    Returns the spawned Entity object.

remove_entity

Remove an entity from the world.

remove_entity(entity) -> Boolean

Returns true if entity was removed.

Example spawning mobs:

func spawn_wave(center_x, center_y, center_z, count) {
    for i in range(0, count) {
        // Random offset from center
        let x = center_x + random(-5, 5);
        let z = center_z + random(-5, 5);
        spawn_entity("minecraft:zombie", x, center_y, z);
    }
}

Environmental Effects

create_explosion

Create an explosion.

create_explosion(x, y, z, power) -> Boolean
create_explosion(location, power) -> Boolean
create_explosion(x, y, z, power, fire) -> Boolean
create_explosion(location, power, fire) -> Boolean
  • power: 0-10 (TNT = 4)
  • fire: Whether to create fire blocks (optional)
    Returns true if explosion was created.

strike_lightning

Create a lightning strike.

strike_lightning(x, y, z) -> Boolean
strike_lightning(location) -> Boolean

Returns true if lightning was created.

Time Control

set_time

Set the world time.

set_time(world, time) -> Boolean
  • time: 0-24000 (0 = sunrise, 6000 = noon, 12000 = sunset, 18000 = midnight)

get_time

Get the current world time.

get_time(world) -> Number

Returns time (0-24000).

Weather Control

set_weather

Change the weather.

set_weather(world, weather, duration) -> Boolean
  • weather: "clear", "rain", or "thunder"
  • duration: Time in ticks
    Returns true if weather was changed.

get_weather

Get the current weather.

get_weather(world) -> String

Returns weather type as string.

World Access

get_world

Get a world by name.

get_world(name) -> World

Returns World object or null if not found.

Broadcasting

broadcast

Send a message to all players in scope.

broadcast(message) -> Boolean

Supports string interpolation with {}.

Examples

Creating an arena:

func create_arena(scope) {
    let region = get_region(scope);
    let x1 = region.x1;
    let x2 = region.x2;
    let z1 = region.z1;
    let z2 = region.z2;
    let y = region.y1;

    // Clear area
    for x in range(x1, x2) {
        for z in range(z1, z2) {
            for dy in range(0, 5) {
                set_block(x, y + dy, z, "minecraft:air");
            }
        }
    }

    // Build floor
    for x in range(x1, x2) {
        for z in range(z1, z2) {
            set_block(x, y, z, "minecraft:stone");
        }
    }

    // Build walls
    for x in range(x1, x2) {
        for dy in range(0, 4) {
            set_block(x, y + dy, z1, "minecraft:stone_bricks");
            set_block(x, y + dy, z2, "minecraft:stone_bricks");
        }
    }

    for z in range(z1, z2) {
        for dy in range(0, 4) {
            set_block(x1, y + dy, z, "minecraft:stone_bricks");
            set_block(x2, y + dy, z, "minecraft:stone_bricks");
        }
    }
}

Day/night cycle event:

OnEvent(TimeChange) {
    let t = get_time(world);

    if t == 0 {
        broadcast("The sun rises!");
    } else if t == 12000 {
        broadcast("Night falls...");

        // Spawn some monsters
        for player in get_online_players() {
            let pos = get_location(player);
            for i in range(0, 3) {
                spawn_entity("minecraft:zombie", pos);
            }
        }
    }
}
