API Reference — Player Functions

API Reference — Player Functions

This page documents all built-in functions for interacting with players in Quill scripts. These functions let you manage player state, inventory, effects, and more.

Movement and Location

teleport

Teleport a player to coordinates or location.

teleport(player, x, y, z) -> Boolean
teleport(player, location) -> Boolean

Returns true if teleport successful.

Example:

// Teleport to coordinates
teleport(player, 0, 65, 0);

// Teleport to location
let spawn = location(0, 65, 0);
teleport(player, spawn);

get_location

Get a player's current location.

get_location(player) -> Location

Returns a Location object with x, y, z coordinates.

Health and Status

heal

Restore a player to full health.

heal(player) -> Boolean

Sets health to maximum (20).

set_health

Set a player's health points.

set_health(player, amount) -> Boolean
  • amount: 0-20 (0 = dead, 20 = full)
    Returns true if health was set.

get_health

Get a player's current health.

get_health(player) -> Number

Returns health points (0-20).

set_hunger

Set a player's hunger level.

set_hunger(player, amount) -> Boolean
  • amount: 0-20 (0 = starving, 20 = full)

get_hunger

Get a player's current hunger level.

get_hunger(player) -> Number

Returns hunger points (0-20).

kill

Kill a player instantly.

kill(player) -> Boolean

Returns true if player was killed.

Game Mode

set_gamemode

Change a player's game mode.

set_gamemode(player, mode) -> Boolean
  • mode: "survival", "creative", "adventure", or "spectator"
    Returns true if mode was changed.

get_gamemode

Get a player's current game mode.

get_gamemode(player) -> String

Returns game mode as string.

Inventory

give

Give items to a player.

give(player, item_id) -> Boolean
give(player, item_id, amount) -> Boolean
give(player, item) -> Boolean
  • item_id: Minecraft item ID (e.g., "minecraft:diamond_sword")
  • amount: Stack size (default: 1)
  • item: Item object from item() constructor
    Returns true if items were given.

Example:

// Give one item
give(player, "minecraft:bread");

// Give stack
give(player, "minecraft:arrow", 64);

// Give custom item
let sword = item("minecraft:diamond_sword", 1, {Enchantments:[{id:"sharpness",lvl:5}]});
give(player, sword);

remove_item

Remove items from a player's inventory.

remove_item(player, item_id) -> Number
remove_item(player, item_id, amount) -> Number

Returns number of items actually removed.

has_item

Check if player has an item.

has_item(player, item_id) -> Boolean
has_item(player, item_id, amount) -> Boolean

Returns true if player has the item(s).

Effects

give_effect

Apply a potion effect to a player.

give_effect(player, effect, duration, amplifier) -> Boolean
  • effect: Potion effect ID (e.g., "speed", "strength")
  • duration: Time in ticks
  • amplifier: Effect level (0 = I, 1 = II, etc.)

remove_effect

Remove a specific effect from a player.

remove_effect(player, effect) -> Boolean

clear_effects

Remove all effects from a player.

clear_effects(player) -> Boolean

Messages

sendmessage

Send a chat message to a player.

sendmessage(player, message) -> Boolean

Supports string interpolation with {}.

sendtitle

Display title text to a player.

sendtitle(player, title, subtitle, fade_in, stay, fade_out) -> Boolean

Times are in ticks (20 ticks = 1 second).

Example:

// Show "Game Over" for 3 seconds
sendtitle(player, "Game Over", "Thanks for playing!", 10, 60, 10);

playsound

Play a sound for a player.

playsound(player, sound, volume, pitch) -> Boolean
  • sound: Minecraft sound ID
  • volume: 0.0-1.0
  • pitch: 0.5-2.0

Movement States

set_flying

Toggle a player's flying state.

set_flying(player, flying) -> Boolean

Only works in creative/spectator mode.

Administration

kick

Kick a player from the server.

kick(player, reason) -> Boolean

Returns true if player was kicked.

Information

get_name

Get a player's username.

get_name(player) -> String

is_online

Check if a player is connected.

is_online(player) -> Boolean

is_op

Check if a player is an operator.

is_op(player) -> Boolean

Examples

Complete example managing player state:

func prepare_for_game(player) {
    // Reset state
    heal(player);
    clear_effects(player);
    set_gamemode(player, "survival");

    // Give starter kit
    give(player, "minecraft:iron_sword");
    give(player, "minecraft:bread", 5);

    // Apply effects
    give_effect(player, "resistance", 100, 0);  // 5 seconds of Resistance I

    // Notify
    sendtitle(player, "Ready!", "Game starting...", 10, 40, 10);
    playsound(player, "entity.player.levelup", 1.0, 1.0);
}
