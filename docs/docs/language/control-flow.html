Language — Control Flow

Language — Control Flow

This page covers Quill's control flow structures that let you make decisions and repeat actions in your scripts: if statements, loops, and control statements.

If Statements

Make decisions based on conditions:

if condition {
    // code runs if condition is true
}

if condition {
    // runs if true
} else {
    // runs if false
}

if condition1 {
    // first check
} else if condition2 {
    // second check
} else {
    // fallback
}

Common condition patterns:

if player.health < 10 {
    heal(player);
}

if in_region(player, arena) && arena.game_active {
    // player is in active game
}

if type_of(item) == "string" {
    log("Got string:", item);
}

For Loops

Three main ways to iterate:

List iteration

for item in list {
    // process each item
}

// Example: Give items to players
let items = ["diamond_sword", "bread", "arrows"];
for item_id in items {
    give(player, item_id);
}

Numeric ranges

for i in range(0, 5) {
    // runs 5 times (0 to 4)
}

// Example: Countdown
for i in range(10, 0) {
    broadcast("Starting in {i}...");
}

Scope players

for player in arena.players {
    // enters arena's context
    if player.health < 20 {
        heal(player);
    }
}

Note: When iterating over scope.players, the code runs in that scope's context and can access its local variables.

While Loops

Repeat while a condition is true:

while condition {
    // repeats until condition is false
}

// Example: Wait for players
arena.min_players = 4;
while len(arena.players) < arena.min_players {
    broadcast("Waiting for more players...");
}

Be careful with while loops:
- Always ensure the condition will eventually become false
- Consider using events instead for continuous checks

Loop Control

Two special statements for controlling loops:

Break

Exit the loop immediately:

for player in arena.players {
    if player.health <= 0 {
        broadcast("Game over!");
        break; // exit loop
    }
}

Continue

Skip to the next iteration:

for player in get_online_players() {
    if !in_region(player, arena) {
        continue; // skip to next player
    }
    // only runs for players in arena
    give(player, "minecraft:diamond");
}

Examples

Game countdown with player checks:

func start_game(arena) {
    arena.starting = true;

    for i in range(10, 0) {
        // Check players each tick
        if len(arena.players) < 2 {
            broadcast("Not enough players, cancelling!");
            break;
        }

        broadcast("Starting in {i}...");
    }

    if arena.starting {
        arena.game_active = true;
        broadcast("Game started!");
    }
}

Processing items with validation:

func give_rewards(player, rewards) {
    for item in rewards {
        // Skip invalid items
        if type_of(item) != "string" {
            log("Invalid item:", item);
            continue;
        }

        // Try to give item
        if !give(player, item) {
            broadcast("Failed to give {item}!");
            break;
        }
    }
}
